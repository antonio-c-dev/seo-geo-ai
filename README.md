# Architettura di un Sito Perfetto — SEO e GEO per l'Era dell'AI

Progetto scolastico di HTML e CSS sulla progettazione di un sito web 
ottimizzato per motori di ricerca (SEO) e motori generativi (GEO).

## Pagine del sito

- [Home — Architettura di un Sito Perfetto](https://seo-geo-ai.netlify.app/)
- [Cos'è la SEO](https://seo-geo-ai.netlify.app/seo.html)
- [Cos'è la GEO](https://seo-geo-ai.netlify.app/geo.html)
- [Caso Studio SEO Reale](https://seo-geo-ai.netlify.app/caso-studio.html)
- [Struttura Tecnica del Sito](https://seo-geo-ai.netlify.app/struttura-tecnica.html)
- [Monitoraggio e Performance](https://seo-geo-ai.netlify.app/contatti.html)

## Tecnologie utilizzate

- HTML5 semantico
- CSS3 con foglio di stile esterno
- Ottimizzazione SEO on-page
- Generative Engine Optimization (GEO)
- Google Search Console
- Sitemap XML e robots.txt
